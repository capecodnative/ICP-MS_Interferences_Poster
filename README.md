# ICP-MS_Interferences_Poster
## A Printable Poster of ICP-MS Spectral Overlaps and Interferences  
Interference data are compiled from numerous references by Lomax-Vogt et al. (2021, Spectrochimica Acta Part B).  
https://doi.org/10.1016/j.sab.2021.106098  
For a full list of intereference references, please see the original Lomax-Vogt publication and references therein.  
   
Atomic abundances listed in 'IsotopeAbundances_CIAAWdogOrg_2024.csv' were compiled by "Isotopic compositions of the elements 2024", available online at www.ciaaw.org and www.ciaaw.org/isotopic-abundances.htm  
For isotopes with an abundance range reported, the mean of the upper and lower range values is used. 
  
Poster creation script by Dan Ohnemus (dan at uga) and coded in Python3 using ChatGPT v5.5
